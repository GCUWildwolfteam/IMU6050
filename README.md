IMU6050
========
使用[STM32F103T8U6](http://www.st.com/en/microcontrollers/stm32f103T8.html)作为主控，[MPU6050](https://www.invensense.com/products/motion-tracking/6-axis/mpu-6050/)作为IMU，通过处理后使用CAN通信把航向角和陀螺仪Z轴数据打包输出
